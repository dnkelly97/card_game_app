This project is a web application built using Rails that provides people a virtual deck of cards, a table, and generic functionality such as drawing and transferring cards. The flexibility of this web application allows user to play many different card games.

This is a copy of the original project repository, which was associated with an organization. The source code is here, but the wiki is lost. To see the project in action, visit https://g003-cardgamesim-sprint3.herokuapp.com/dashboard.
